# Django Portfolio Website

This is a professional portfolio website built using Django.

## Features
- User authentication (register, login, logout)
- Login-protected home page
- Projects and project detail pages
- Admin dashboard to manage projects
- Contact and feedback system
- Resume download feature

## Technologies Used
- Python
- Django
- HTML
- CSS
- SQLite

## How It Works
- Users must log in to access the website
- New users register first, then log in
- Projects are displayed on the home and projects pages
- Admin can add, edit, and delete projects
- Users can send messages via contact and feedback forms

## Setup
1. Clone the repository
2. Install Django
3. Run migrations
4. Create superuser
5. Run server
